# Task API Logging

REST API for managing tasks with error handling and logging

This project demonstrates:
- Structured logging
- Global exception handling
- HTTP error responses

## Tech Stack
Java 21 · Spring Boot 3.5.12 · Maven · Docker 

## Getting Started

1. Clone repo
```bash
git clone https://github.com/noagsa/task-api-logging
cd task-api-logging
```
2. Execute application
```
./mvnw spring-boot:run
```
